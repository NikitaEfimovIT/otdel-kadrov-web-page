# Дизайн-спека сайта гильдии «Отдел кадров» — для Claude Design

> Бриф для генерации дизайна. Отдай его Claude Design целиком вместе с файлом герба.
> Всё, что помечено `[уточнить]`, замени перед финальной генерацией.

---

## 1. О проекте

Сайт гильдии World of Warcraft «Отдел кадров». Решает две задачи одновременно:

- **Визитка (внешняя):** привлекать и отбирать новых игроков — показать гильдию лицом, дать быстрый доступ к логам и raider.io, объяснить, кого и на что мы ищем.
- **Хаб (внутренняя):** инструмент коммуникации для своих — новости, опросы, прогресс рейда, единая точка со всеми ссылками.

Формат: адаптивный сайт (desktop + mobile), тёмная тема по умолчанию. Русский язык интерфейса.

## 2. Аудитория и позиционирование

Гильдия — **полу-хардкор с человеческим лицом**: берём кромку в рейдах (Heroic, заход в Mythic) и закрываем высокие ключи M+, но без токсичности и пафоса. Ровно это должен считывать и рекрут, и свой:

- **Рекрут:** «эти ребята умеют, но с ними комфортно». Виден прогресс и требования, но нет вайба элитарного гейткипинга.
- **Участник:** «здесь удобно» — новости и опросы под рукой, ничего лишнего.

Настроение: уверенно, дружелюбно, с лёгкой самоиронией.

## 3. Бренд

Ядро айдентики — **герб**: геральдический щит, синее поле, золотая кайма, золотой драконий мотив (три головы) внутри. Название гильдии в игре отображается зелёным — это можно обыграть как акцент.

**Метафора и тональная фишка.** Название «Отдел кадров» контрастирует с эпичным драконьим гербом — это и есть характер гильдии: серьёзный рейд-кред плюс самоирония. Рекомендую аккуратно обыграть «корпоративно-кадровую» тему поверх эпик-фэнтезийной базы (не переусердствовать — рейд-достижения подаём серьёзно):

- Рекрутинг → раздел «Вакансии» / «Мы нанимаем», кнопка «Отправить резюме» вместо «Вступить».
- Ростер → «Штат» / «Сотрудники».
- Новости → «Приказы» / «Доска объявлений».
- Опросы → «Планёрка».
- Прогресс рейда → «Отчётность» / «KPI».

Это опциональный слой копирайта — визуально гильдия остаётся эпичной, юмор живёт в подписях.

## 4. Палитра

Выведена из герба (синий + золото), тёмная тема. Все пары проверены на контраст WCAG 2.1 AA (см. §11).

| Токен | HEX | Назначение |
|---|---|---|
| `bg` (фон страницы) | `#0E1B33` | Глубокий тёмно-синий — основной фон |
| `surface` (панель/карточка) | `#16294D` | Карточки, секции, панели |
| `brand-blue` | `#21467F` | Королевский синий из щита — бейджи, акценты, заливки |
| `gold` | `#E3B23C` | Основное золото — заголовки, рамки, кнопки, разделители |
| `gold-bright` | `#F1C860` | Яркое золото — hover, свечение, активные состояния |
| `text` | `#EAF0FA` | Основной текст (почти белый, холодный) |
| `text-muted` | `#9FB0CE` | Второстепенный текст, подписи, даты |
| `guild-green` | `#47D147` | Акцент под цвет имени гильдии в игре — вордмарк, статусы «онлайн»/«успех» |

Правило использования: золото — для иерархии и «геральдики» (заголовки, рамки, ключевые CTA), не заливать им большие плоскости. Зелёный — точечно.

## 5. Типографика

- **Заголовки:** шрифт с характером, геральдический/фэнтезийный, но с поддержкой кириллицы и хорошей читаемостью. Направление: Cinzel / Marcellus / Philosopher (обязательно проверить кириллицу — Philosopher её поддерживает).
- **Текст и UI:** чистый гротеск — Inter / Manrope. Отличная читаемость, полная кириллица.
- Иерархия: крупные золотые H1/H2, светлые H3, `text-muted` для подписей.
- Числа прогресса и статистики — можно моноширинным акцентом для «отчётного» вида.

## 6. Визуальный язык

- **Геральдика дозированно:** тонкие золотые линии-разделители, угловые орнаменты у секций, герб как деликатный водяной знак в hero. Без тяжёлых текстур и «золотого барокко» — фон чистый, акценты точечные.
- **Карточки:** `surface` с тонкой золотой обводкой (1px, полупрозрачное золото), мягкое золотое свечение при наведении, скругление 8–12px.
- **Иконки:** линейные, в стиле lucide, золотые/светлые.
- **Изображения:** скриншоты рейдов/боссов допустимы как фон hero с тёмным сине-градиентным оверлеем, чтобы текст оставался читаемым.
- **Глубина:** аккуратные тени в синеву, лёгкие градиенты `bg → surface`. Никакого неона.

## 7. Структура сайта

Одностраничник с якорной навигацией (при росте контента новости/опросы можно вынести на отдельные страницы).

1. **Шапка / навигация** — sticky, тёмная; слева герб + вордмарк «Отдел кадров», справа пункты меню, активный пункт золотом.
2. **Hero** — герб, название, слоган, ключевые CTA.
3. **О гильдии + прогресс рейда** — позиционирование и трекер боссов / рейтинг M+.
4. **Новости** — лента постов.
5. **Опросы** — активный опрос + архив.
6. **Ссылки** — хаб внешних ресурсов.
7. **Рекрутинг** — кого ищем, требования, форма заявки.
8. **Footer** — Discord/соцсети, копирайт, «Powered by Raidsmith» (на будущее).

## 8. Секции лендинга — детально

**Hero.** Тёмно-синий фон (опц. приглушённый скриншот рейда под оверлеем), по центру или слева — герб. Крупный золотой вордмарк «Отдел кадров», под ним слоган (например: «Бьём кромку, закрываем ключи, ржём в войсе»). Две кнопки: основная золотая «Отправить резюме» (заявка) и вторичная-контурная «Наши логи». Мелким — регион/сервер `[уточнить]` и фракция.

**О гильдии + прогресс.** Слева короткий текст-позиционирование (2–3 предложения). Справа — **трекер прогресса рейда**: название актуального рейда, счётчики `X/Y Normal · X/Y Heroic · X/Y Mythic` с золотыми прогресс-барами, плюс блок M+ (лучший ключ / рейтинг). Статус боссов — иконкой + подписью (не только цветом).

**Новости.** Заголовок секции + сетка карточек (дата, заголовок, превью, тег). На десктопе 2–3 колонки, на мобиле — одна. Кнопка «Все новости».

**Опросы.** Карточка активного опроса: вопрос, варианты с золотыми полосами-результатами и процентами, кнопка голоса; состояние «уже проголосовано». Ниже — свёрнутый архив прошлых опросов.

**Ссылки.** Ряд крупных кнопок-плиток с иконками: raider.io, Warcraft Logs, Discord, WoWProgress, YouTube/Twitch — `[дай финальный список ссылок]`. Каждая плитка — `surface` с золотой обводкой и логотипом сервиса.

**Рекрутинг.** Блок «Кого ищем»: нужные классы/роли (карточки ролей — танк/хил/дд), требования (мин. рейтинг, готовность к прогрессу, микрофон). Кнопка/форма заявки: имя-BattleTag, класс/спек, ссылка на logs/raider.io, комментарий. В «кадровом» тоне — как отклик на вакансию.

**Footer.** Герб мелким, ссылки на соцсети, копирайт, строка «Скоро: интеграция с Raidsmith».

## 9. Ключевые компоненты

- **Навбар** — sticky, тёмный, золотой активный пункт; на мобиле бургер.
- **Карточка новости** — дата (`text-muted`), заголовок (золото), превью, тег.
- **Виджет опроса** — варианты, прогресс-бары, %, состояния «до/после голоса».
- **Трекер прогресса рейда** — счётчики боссов по сложностям + бары; блок M+.
- **Плитка-ссылка** — иконка сервиса, подпись, hover-свечение.
- **Карточка роли** (рекрутинг) — иконка роли, класс, статус «нужен/закрыт».
- **Форма заявки** — поля + валидация, кнопка отправки.
- **Кнопки** — primary (золотая заливка, тёмный текст), secondary (контур золотом), ghost.

Для каждого интерактива задать состояния: default / hover / focus / active / disabled.

## 10. Адаптивность

Mobile-first. Брейкпоинты: 640 / 768 / 1024 / 1280px.

- Сетки карточек: 3–2 колонки на десктопе → 1 на мобиле.
- Hero: горизонтальная композиция → вертикальная; CTA на всю ширину.
- Навигация → бургер-меню на < 768px.
- Кликабельные зоны ≥ 44×44px.

## 11. Доступность (проверено)

Контраст пар палитры на тёмном фоне (WCAG 2.1 AA требует ≥ 4.5:1 для обычного текста, ≥ 3:1 для крупного):

- Светлый текст `#EAF0FA` на фоне `#0E1B33` — **15.0:1**
- Светлый текст на панели `#16294D` — **12.6:1**
- Приглушённый `#9FB0CE` на фоне — **7.8:1**
- Золото `#E3B23C` на фоне — **8.8:1**
- Тёмный текст `#0E1B33` на золотой кнопке — **8.8:1**
- Зелёный `#47D147` на фоне — **8.6:1**

Все пары проходят AA с запасом. Дополнительно: не кодировать статусы только цветом (боссы killed/alive, «нужен/закрыт» — иконка + текст); видимый focus-ринг золотом; альты для герба и логотипов.

## 12. Тон UX-copy

Русский, обращение на «ты», уверенно и по-доброму, с лёгкой кадрово-корпоративной самоиронией (см. §3). Без канцелярита и без элитарного гейткипинга.

Примеры:
- Слоган: «Бьём кромку, закрываем ключи, ржём в войсе».
- CTA рекрутинга: «Отправить резюме».
- Пустое состояние опросов: «Планёрки пока нет. Отдыхаем».
- Заголовок рекрутинга: «Открыт набор в штат».

## 13. Задел под Raidsmith (на будущее)

Сайт позже интегрируется с Raidsmith. Заранее заложить в макет:

- Место под виджеты Raidsmith в блоке прогресса (авто-подтягивание ростера, посещаемости, логов).
- Кнопку «Войти через Raidsmith» в шапке (скрыта/задизейблена до релиза интеграции).
- Строку «Powered by Raidsmith» в футере.

Пока это статические заглушки — данные вводятся вручную.

## 14. Как использовать с Claude Design

1. Приложи этот файл + изображение герба (для точной подгонки золота и синего под оригинал).
2. Приоритет для первой генерации: **Hero → Прогресс рейда → Новости** (это лицо сайта). Опросы, ссылки, рекрутинг — вторым проходом.
3. Проси тёмную тему, геральдические акценты дозированно, палитру из §4, компоненты из §9.
